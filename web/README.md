# ⛳ スイング動画編集ツール(Web版)

ゴルフスイング動画を読み込むと、スイング部分を自動検出して
「1回目: 通常速度 → 2回目: スロー再生(赤いガイド線付き)」の動画を作成するツールです。
複数の動画を読み込むと1本に連結されます。

**動画はどこにもアップロードされません。** すべての処理(スイング検出・姿勢推定・動画変換)は
お使いの端末のブラウザ内で完結します。

## 使い方

1. ページを開き、動画をドラッグ&ドロップ(スマホはタップして選択)
2. スイング区間が自動検出される(ズレていたら±ボタンで微調整)
3. 「💾 動画を作成する」を押し、完了したら「⬇ 完成動画を保存」

## ガイド線

- **正面から撮った動画**: アドレスの鼻の位置に縦の赤線
- **後方から撮った動画**: シャフトの線 / ボールと首の付け根を結ぶ線 / 頭〜お尻の背中の線
- 撮影方向は自動判別(一覧のプルダウンで手動指定も可能)

## 対応環境

- iPhone / iPad: Safari (iOS 16.4以降)
- Android: Chrome
- PC: Chrome / Edge / Safari

## 技術構成

- スイング検出: フレーム差分の動き解析 + インパクト音検出(Web Audio API)
- 姿勢推定: [MediaPipe Pose Landmarker](https://ai.google.dev/edge/mediapipe/solutions/vision/pose_landmarker) (ブラウザ内で実行)
- 動画書き出し: Canvas + MediaRecorder API
